#Spot Ninja

[Spot Ninja](https://spot-ninja.firebaseapp.com/)

[Spot Ninja API](https://spotninja.herokuapp.com/)

##Getting Started

```sh
./setup.sh
nodemon
```

##Routes

####GET all spots:
/spots

####GET all users:
/users

####GET single user:
/spots/:id

####GET user's spots:
/users/:id

####POST new spot:
/spots/new

####POST new user:
/users/new

####PUT update spot:
/spots/at/:id

####DELETE single spot:
/spots/:id
