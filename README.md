#Spot Ninja

[Spot Ninja API](https://spotninja.herokuapp.com/)

##Getting Started

```sh
./setup.sh
nodemon
```

##Routes

####GET all spots:
/spots

####GET all users:
/users

####GET single spot:
/spots/:id

####GET spots by user:
/spots/user/:id

####POST new spot:
/spots/new

####POST new user:
/users/new

####PUT update spot:
/spots/:id

####DELETE single spot:
/spots/:id
