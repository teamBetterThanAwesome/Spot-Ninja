#Spot Ninja

[Spot Ninja](https://spotninja.herokuapp.com/)

##Getting Started

```sh
./setup.sh
nodemon
```

##Routes

####GET all spots:
/spots

####GET single spot:
/spots/:id

####GET spots by user:
/spots/user/:id

####POST new spot:
/spots

####PUT update spot:
/spots/:id

####DELETE single spot:
/spots/:id
